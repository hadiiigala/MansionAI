How to Generate Results

  1. Run the CLIP similarity evaluation: 
    python clip_similarity.py
  2. Generate visualizations: 
    python clip_eval_visualizer.py
  3. Refresh this page to see the results

Notes

  • Results will be populated after running the evaluation scripts
  • Visualizations require matplotlib and seaborn to be installed
  • The similarity scores are cosine similarities between CLIP embeddings (0 to 1, where 1 is most similar)
  • If images don't appear, check that the visualization files were generated successfully
  • The visualizer automatically handles cases where there are fewer image pairs than requested
